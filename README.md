ThriftKart 

ThriftKart is a full-stack Java web application built with Spring Boot, Thymeleaf, and Maven, designed to simulate a thrift store for pre-loved items and homemade goodies. The application allows users to browse products, manage carts, process orders, and interact with support—all through a clean, responsive web interface.
Note: This project was built as a part of Semester 3 CSE Project-Based Learning.

Features
Product Catalog – Browse and search pre-loved items and homemade products.
Shopping Cart – Add, remove, or update products in your cart.
Order Management – Place orders and view order history.
Support System – Submit and track support tickets.
Admin Dashboard – Manage products, orders, and analytics (optional if implemented).
Responsive UI – Clean interface powered by Thymeleaf templates.

Technologies Used
Backend: Java, Spring Boot
Frontend: Thymeleaf, HTML, CSS, Bootstrap
Build Tool: Maven
Database: H2 / MySQL (configurable)
Version Control: Git & GitHub

Getting Started
Prerequisites
Java JDK 17+
Maven 3.8+
(Optional) MySQL or other relational database if not using H2
